## 개요
가자맵은 카카오 맵 SDK V2를 활용하여 영업사원과 같은 위치 정보가 중요한 직업군을 위해 설계된 고객 관리 앱입니다.  
고객의 위치를 시각적으로 표시하고, 현재 위치를 기준으로 반경 내 고객을 필터링하여 효율적인 관리가 가능하도록 돕습니다.
